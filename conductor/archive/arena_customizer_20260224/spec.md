# Visual Arena Customizer (AI-Assisted) - Specification

## Overview

Allow Bookers to define the "look and feel" of an event, with AI generating unique background images and influencing crowd atmosphere.

## Features

- **Theme Selection:** Define themes like *Cyberpunk*, *Old School*, *Outdoor*.
- **AI Image Generation:** Generate arena backgrounds based on selected theme.
- **Crowd Atmosphere:** Visuals influence a "Crowd Heat" modifier.
- **Dashboard Integration:** Apply generated art to the show dashboard.
