**Status:** Completed

**Start Date:** 2026-01-29

## Overview

This track implements the foundational features for multiplayer interaction within the All Time Wrestling RPG. The primary goals are to allow Bookers to draft wrestlers into their promotions (Draft Mode) and to allow Viewers (or the public) to view the history of shows produced by other users (Public Showcases).
