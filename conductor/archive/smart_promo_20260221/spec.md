# AI-Driven "Smart Promo" Response System - Specification

## Overview

Enhance interactive promos by having the AI offer the player 3-4 distinct "Rhetorical Hooks" and generating context-aware responses.

## Features

- **Rhetorical Hooks:** AI generates options like *Insult the City*, *Challenge Honor*, *Pander to Fans*.
- **Context Awareness:** Responses reference recent match history and rivalries.
- **Alignment Impact:** Success depends on alignment (Face/Heel) consistency.
- **Dynamic Feedback:** Crowd reaction changes based on performance.
