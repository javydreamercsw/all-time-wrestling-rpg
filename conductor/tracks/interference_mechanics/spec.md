# Match Interference & Manager Mechanics - Specification

## Overview

Create a system for non-participating wrestlers to influence matches. Allies or Managers can "distract the ref" or "provide a weapon" via ability cards.

## Features

- **Manager/Ally Slot:** Assign non-wrestlers to ringside.
- **Interference Cards:** New ability cards specifically for interference.
- **Ref Distraction:** Mechanic to negate pin attempts or allow illegal moves.
- **Ejection Risk:** Chance for the manager to be ejected from ringside.
