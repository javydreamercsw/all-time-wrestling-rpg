# Booker's Budget & Economic Simulation - Specification

## Overview

Add a financial management layer for the BOOKER role. Shows would have costs (venue rental, pyrotechnics, talent fees) and revenue (ticket sales based on hype, merchandise).

## Features

- **Show Costs:** Venue, Pyro, Talent Fees, Advertising.
- **Show Revenue:** Ticket sales based on *Show Hype* and *Main Event Draw*.
- **Wrestler Contracts:** Appearance fees based on Fan Ranking.
- **Financial Dashboard:** Monthly/Yearly P&L statement.
- **Bankruptcy:** Potential game over or restart if funds run dry.
