# Dynamic Faction Chemistry & Synergies - Specification

## Overview

Expand the faction system so that being part of a stable like "Team USA" or "Desolation's Smile" provides tangible gameplay bonuses.

## Features

- **Affinity Points:** Members gain points when appearing in the same segment.
- **Shared Buffs:** High affinity leads to passive buffs (e.g., +1 Momentum per turn when ally is ringside).
- **Double-Team Moves:** Unlock unique AI narration triggers for double-team finishers.
- **Faction Management UI:** View affinity levels and active bonuses.
